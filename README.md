# 🤖 Telegram Chat Bot

A simple and customizable **Telegram Bot** built with **Node.js**. This bot can be extended to include commands, moderation, and interactive features for Telegram groups and personal chats.

---

## 🚀 Features

- 💬 Responds to user messages
- ⚡ Built with **Telegraf.js** for Telegram Bot API
- 🛡️ Extendable with filters, moderation, and commands
- 🌐 Can be hosted locally or on cloud servers

---

## 📂 Project Structure

```
TelegramChatBot-main/
│── TelegramBot.js       # Main bot script
│── package.json         # Dependencies & metadata
│── package-lock.json    # Lock file
```

---

## ⚙️ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/AkshayBakki5/Telegram-Chat-Bot.git
   cd TelegramChatBot-main
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Set up environment variables**
   Create a `.env` file and add your **Telegram Bot Token**:

   ```env
   API_KEY=your-telegram-bot-token-here
   ```

4. **Run the bot**

   ```bash
   node TelegramBot.js
   ```
